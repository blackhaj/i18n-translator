# i18n Translator

It's a bore creating i18n translations for each bit of text of your website. This script automates the process using translate (not perfect, I know).

**Prerequisites**
You need to set up and account with Google Cloud to access the translator API. Once you have set up a project and downloaded your authentication credentials, you need to add the as an environment variable:
`export GOOGLE_APPLICATION_CREDENTIALS="[PATH TO CREDENTIALS]"`

Docs here - https://cloud.google.com/docs/authentication/getting-started
